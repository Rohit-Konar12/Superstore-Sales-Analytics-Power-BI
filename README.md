# Superstore-Sales-Analytics-Power-BI
Interactive Power BI dashboard built from cleaned Superstore sales data using Python for data cleaning, EDA and business insights.

# Project Overview
This project present an end-to-end sales analytics solution using the Superstore dataset. The workflow include data cleaning, exploratory data analysis (EDA), feature engineering, and the development of an interactive Power BI dasboard to uncover actionable business insights.

The dashboard enable users to analyze sales performence, profitability, customer segments, regional trends, and operational metrics through visualization and KPI cards.

# Project Objectives
>Clean and preprocess raw sales data for analysis.
>Perform exploratory data analysis to identify business trends.
>Create meaningful feature to improve reporting.
>Build an interactive Power BI dashboard for business decision making.
>Generate actionable insights from sales and profit data.

# Data Cleaning & Feature Engineering
The following preprocessing steps were performed before creating the dashboard:
> Remove duplicates recird.
> Checked and handled missing values.
> Corrected inconsistent data types.
> Standarized column names.
> Create new features:
    > Delivery Days
    > Order Year
    > Order Month
    > Month Name
    > Order Day
> Exported the cleaned dataset for Power BI visualization.

# Exploratory Data Analysis(EDA)
EDA was performed to understand the business performence and identify trends, including:
> Sales trend over time.
> Profitability analysis.
> Category and Sub-category analysis.
> Customer segment analysis.
> Regional sales comparision.
> Discount impact on profit.

# Business Insights
> Sales showed a stedy upward trend between 2014 and 2017
> The Consumer segment generated the highest sahre of overall sales.
> Technology was most profitable product category.
> Phones and Chairs were among the highest selling sub-category.
> Regional performence varied significantly, highlighting opportunities for targeted business strategies.
> Fast delivery performence contributed to efficient order fullfillment.
